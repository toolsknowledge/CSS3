CSS
---
    - CSS Stands for Cascadding Style Sheet.

    - CSS, used to apply the styles to the webpages.

        Ex.
            borders

            colors

            background-colors

            font styles

            shadows

            animations

            transformations

            ----
            ----
            ----

    - these are the different versions of CSS

            CSS 1.0

            CSS 2.0

            CSS 3.0

    - current version is CSS 3.0

    - <style></style> tag, used to include the CSS to webpages.

        Ex.
            <style>
                //CSS Here
            </style>

    - "style" attribute, used to write the "inline CSS".

        Ex.
            <div style="......"></div>

    - <link> tag, used to include the external CSS.

    - CSS files must save with ".css" extension.
